# Volume Limiter  
本应用可以在后台自动降低系统音量。  

## 使用技巧  
将快捷方式放入`C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup`文件夹，可以实现开机自启。  
在程序目录下创建`MaxVolume.txt`文件，填入0~100的整数，可以自定义最大音量限制。文件不存在或内容非法时，使用默认限制(`30%`)。  
本程序依赖`.NET 10运行时，需要预先安装。
